# Create a component for Drupal and Angular

<br/>

### 1. Duplicate components/example folder

### 2. Create your component 
- `npm run generate component <component-name>`

### 3. Make sure component is
- Accessible (use [Pa11y](https://github.com/pa11y/pa11y) => `npm run test`)
- Agnostic/Universal
- Content resilient
- Responsive
- Using design tokens from _variables.scss
- API is consistent with other components

### 4. Export component
- Export Module and/or Component files in `/lib/public-api.ts`.

### 4. Build library
- Increment the version in `package.json`.
- In terminal, go to the root of the repo.
- `npm run build --prod`
- `npm publish dist/front-end-library`
- Optionnal: Check the registry (TODO: add link)

### 5. Optionnal: Test implementation
- Go to `projects/angular-example`
- `npm i bifrost-front-end-library`
- You can now import the new component.
