- ...

### Install the package
In the Drupal Theme.
```Shell
npm install bifrost-front-end-library --save
```

### Create Namespace @components
Shortcut to simplify path to components:
https://symfony.com/doc/4.1/templating/namespaced_paths.html

### Include components in your Twig file
```js
{% include '@components/button/button.twig' with { size:'lg', color:'primary' } %}
```

**API documentation is provided in Storybook:**
- Open the `Drupal` section for each component.
- Take a look at the `Controls` panel

<br/>

### Add base style
Add `base.min.css` to the main HTML file:

```html
<head>
    <link href="node_modules/bifrost-front-end-library/base.min.css" rel='stylesheet'>
</head>
```

### Optionnal: Access global SASS variables
From your SASS file, import the `_variables.scss`.
```css
@import 'node_modules/bifrost-front-end-library/global/_variables.scss'

body {
    background-color: $primary;
}
```
