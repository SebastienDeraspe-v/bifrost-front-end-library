## Migration
In order to avoid visual regression, we could use **[backstopjs](https://github.com/garris/BackstopJS)**. This will create snapshots of your website and compare the diff.
Note:
- You enter all urls manualy.
- You can reproduce user action (hover, click, ...).
- Snapshots are not display on a public URL. But can be retrieve in GIT.
[Integration Drupal and BackstopJS](https://medium.com/limoengroen/integrating-backstopjs-in-our-drupal-updates-workflow-part-1-683da08bb629)

## 1. Install the package
In the Drupal Theme.
```Shell
npm install bifrost-front-end-library --save
```

## 2. Create Namespace @components
[Install components module](https://www.drupal.org/project/components) in Drupal. This will create **namespace/shortcut** to the NPM Package.
Then, add theses paths to **your-theme.info.yml**:
```
component-libraries:
  components:
    paths:
      - node_modules/bifrost-front-end-library/source/components/
  organisms:
    paths:
      - node_modules/bifrost-front-end-library/source/organisms/
  templates:
    paths:
      - node_modules/bifrost-front-end-library/source/templates/
```

## 3. Include components
You are now ready to include bifrost elements in your Twig files (E.g. Add theses lines to **templates/page.html.twig**):
```html
<!-- Include component -->
{% include '@components/button/twig/button.twig' with { 
    label:'This is a button',
    color:'secondary'
} only %}

<!-- Include organism -->
{% include '@organisms/hero/twig/hero.twig' with { 
    title:'Hero title'
} only %}

<!-- Include template -->
{% include '@templates/contact-form/twig/contact-form.twig' with { 
    title:'Contact form title'
} only %}
```

> **Don't forget to clean Drupal cache:** Configuration > Performance > Clean all cache.

## 4. Add style

? Add `base.min.css`. This will provide the same CSS base for all Videotron websites.

Add `style.min.css`:
<br/>

### Option 1:
Create a library **your-theme.libraries.yml**:
```js
global-styling:
  css:
    theme:
        styles.css: {}
        'node_modules/bifrost-front-end-library/source/style.min.css': {}
```

Then, add this library to **your-theme.info.yml**:
```js
libraries:
  - your-theme/global-styling
```

### Option 2:
In your CSS file:
```css
@import 'node_modules/bifrost-front-end-library/source/styles/style.min.css';
/* or */
@import '~bifrost-front-end-library/source/styles/style.min.css';
```

### Option 3:
In the **head** of your pages:
```html
<head>
    <link href="node_modules/bifrost-front-end-library/source/styles/style.min.css" rel='stylesheet'>
</head>
```

## 5. _Optional:_ Access global SASS variables
From your SASS file, import the `_variables.scss`.
```css
@import 'node_modules/bifrost-front-end-library/source/globals/_variables.scss'

body {
    background-color: $primary;
}
```
