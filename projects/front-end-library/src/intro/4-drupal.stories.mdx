# Drupal implementation
You have a **Drupal installation** and you want to use the Design System.

<br/><br/>

## Requirements
- **Twig v1.x** (impossible to upgrade...)
- **Drupal** requirements?
- **Simphony** requirements?
- ...

## 1. Install the package
In the Drupal Theme.
```Shell
npm install bifrost-front-end-library --save
```

## 2. Create Namespace @components
Shortcut to [simplify path to components](https://symfony.com/doc/4.1/templating/namespaced_paths.html).

## 3. Include components in your Twig file
```js
{% include '@components/button/button.twig' with { size:'lg', color:'primary' } %}
```

**API documentation is provided in Storybook:**
- Open the `Drupal` sub-page for each component.
- Open the `Canvas` Tab and take a look at the `Controls` panel.

## 4. Add base style
Add `base.min.css` to the main HTML file:

```html
<head>
    <link href="node_modules/bifrost-front-end-library/base.min.css" rel='stylesheet'>
</head>
```

## 5. _Optional:_ Access global SASS variables
From your SASS file, import the `_variables.scss`.
```css
@import 'node_modules/bifrost-front-end-library/source/globals/_variables.scss'

body {
    background-color: $primary;
}
```
