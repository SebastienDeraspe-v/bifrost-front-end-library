## Type variant
The type is changed by adding `type="Small/Large/Icon"`.<br/>
Use `icon="NameOfIcon"` to add icon to the button.
**It won't appear with small type.**<br/>
The default type is `small`

## FullWidth
The fullWidth style is automatic on mobile devices but can be used anytime by adding `fullWidth="true"`
